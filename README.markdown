簡介
----------
本專題為可調天線方向之三頻中繼器，內部軟體如下，運用arduino與MIT inventory製作程式，結合手機APP藉以實踐改變天線方向。


開發軟體
----------
* [Codeblocks Arduino IDE (Release 20140626)]
* [EAGLE 7.4.0]
* [EAGLE Library-RBBB] (http://moderndevice.com/rbbb-library)
* [MIT inventory 2]



手機APP
----------
* [APK](APP/version1.0.0.apk)
